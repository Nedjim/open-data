# Open Data - Parcs et Jardins de Paris


## 1 - Présentation

Cette application indique les parcs et jardins de Paris.

## 2 - Execution

```
$ npm install
$ npm start
```
Aller à l'adresse: http://localhost:X

## 3 - Technologies
- API: Google Places
- Reactjs
- Webpack
- Javascript / ES6
- Sass
